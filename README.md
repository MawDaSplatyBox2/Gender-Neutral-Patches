# Gender Neutral Patches
Repository for various gender-neutral patches of popular mods. Most of these patches use GNMTokens.
## AGE Gender Neutral
Patches dialogue and events from Adventurer's Guild Expanded. Has additional support for Romanceable Rasmodia.
## Mr. Ginger Gender Neutral
Patches dialogue and events from Mr. Ginger.
## RSV No Ser
Patches Phillip and Yuuma's dialogue & events from Ridgeside Village. Replaces all instances of "ser" with an alternative word of the player's choice.
